# Currency Converter 

**Currency Converter** — это простой веб-конвертер валют, который позволяет быстро пересчитать одну валюту в другую через удобный, минималистичный и красивый интерфейс 

## Функциональность
- Конвертация между различными валютами
- Быстрая и интуитивная работа прямо в браузере
- Адаптивный дизайн для разных экранов
- Лёгкая настройка курсов валют в JS
  

## Технологии
- **HTML** — структура страницы  
- **CSS** — стилизация интерфейса  
- **JavaScript** — логика конвертации
